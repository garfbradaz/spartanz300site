# Spartanz 300 Images

This folder contains image assets for the Spartanz 300 website.

## Image Guidelines

- Optimize images for web before adding
- Use descriptive filenames (e.g., `clan-logo.png`, `member-avatar-john.jpg`)
- Prefer WebP format for better compression
- Include @2x versions for retina displays when needed

## Recommended Sizes

- Hero backgrounds: 1920x1080px
- Member avatars: 300x300px
- Logo: 200x200px (PNG with transparency)
